Support

Issue tracker

https://github.com/mpc-qt/mpc-qt/issues

Got a question, wish to report a bug, or request a new feature? Visit the issue tracker and open an issue. Try to determine if the issue already exists before you open a new issue. Remember to be courteous.

 

Reporting bugs

Please state the version of Mpc-Qt that you are using, followed by a description of the bug.

If you have a sample file, these can be uploaded to Google Drive, any of the pomf clones, or another suitable service. Alternatively you may wish describe what the file is called and where to download it.
